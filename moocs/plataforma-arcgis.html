Aumentando la productividad
con ArcGIS

INSCRÍBETE AHORA

Duración: 3.5h
Precio: Gratis
Lecciones: 31

Descripción

Este MOOC está dirigido a desarrolladores web, especialmente a aquellos que ya conocen la plataforma ArcGIS pero que quieren actualizarse.

Este curso se compone de pequeñas lecciones en vídeo en las que el alumno podrá refrescar sus conocimientos sobre la plataforma y las piezas clave que la hacen única.

Al final de cada lección se incluye un pequeño test de auto-evaluación para que el alumno pueda verificar que ha comprendido los aspectos más importantes.

Contenidos

Módulo Lección Detalles
Módulo 1:
Introducción 		Objetivos del curso Veremos a quién va dirigido este curso, requisitos recomendados, objetivos y metodología.
Cuenta de organización Repaso a los tipos de cuentas, qué incluyen, qué ocurre cuando expiran, créditos y coste.
Documentación Recursos oficiales, documentación de la API JS y otros recursos.
Comunidad ArcGIS Dónde conectar con otros profesionales, actividad de la comunidad y otras referencias.
Módulo 2:
Web maps & Web scenes 		Introducción a los web maps & scenes Qué son, dónde se almacenan, metadatos, datos contenidos, por qué son importantes, ventajas a nivel de desarrollo y otras consideraciones.
Manipular web maps Cómo se pueden crear, editar y consumir: interfaces gráficas, librerías, APIs y Runtimes.
Especificación del web map Repaso de la estructura en detalle del formato JSON.
Web map como modelo de información Analizamos el web map como modelo de información geográfico junto al ecosistema de utilidades para desarrolladores que le rodean.
Consumir web maps / scenes usando JS Veremos cómo podemos cargar el modelo (sin pintarlo) además de cómo cargarlo usando las clases: WebMap y WebScene.
Cuándo utilizarlos Descubriremos las situaciones en las que el web map son especialmente útiles.
Consejos y trucos Trucos y consejos para depurar el código, acelerar el desarrollo, gestionar tus datos, etc.
