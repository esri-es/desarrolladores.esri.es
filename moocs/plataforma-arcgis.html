Descripción

Si eres un desarrollador web, has creado aplicaciones usando la API de ArcGIS para JavaScript y quieres mejorar tu productividad, este MOOC es para ti.

Metodología: Este curso se compone de pequeñas lecciones en vídeo con un pequeño test de auto-evaluación al final de cada módulo.

Este curso NO tiene como objetivo enseñarte a:

  • Crear aplicaciones desde cero usando la API de ArcGIS para JavaScript
  • Gestionar geoservicios web
  • Realizar análisis espacial
  • Automatizar ni gestionar servidores
  • etc.

Si buscar formarte en estos u otros aspectos te recomendamos que visites:

Programa experto en desarrollo GIS Cursos de formación de Esri España

Contenidos

Módulo Lección Detalles Ver
Módulo 1:
Introducción 		Objetivos del curso Veremos a quién va dirigido este curso, requisitos recomendados, objetivos y metodología.
Cuenta de organización Repaso a los tipos de cuentas, qué incluyen, qué ocurre cuando expiran, créditos y coste.
Documentación Recursos oficiales, documentación de la API JS y otros recursos.
Comunidad ArcGIS Dónde conectar con otros profesionales, actividad de la comunidad y otras referencias.
Módulo 2:
Web maps & Web scenes 		Introducción a los web maps & scenes Qué son, dónde se almacenan, metadatos, datos contenidos, por qué son importantes, ventajas a nivel de desarrollo y otras consideraciones.
Manipular web maps Cómo se pueden crear, editar y consumir: interfaces gráficas, librerías, APIs y Runtimes.
Especificación del web map Repaso de la estructura en detalle de la especificación en formato JSON.
Web map como modelo de información Analizamos el web map como modelo de información geográfico junto al ecosistema de utilidades para desarrolladores que le rodean.
Consumir web maps / scenes usando JS Veremos cómo podemos cargar el modelo (sin pintarlo) además de cómo cargarlo usando las clases: WebMap y WebScene.
Cuándo utilizarlos Descubriremos las situaciones en las que el web map son especialmente útiles.
Consejos y trucos Trucos y consejos para depurar el código, acelerar el desarrollo, gestionar tus datos, etc.
Módulo 3:
Apps Configurables 		Introducción a las apps configurables Veremos qué son, para qué se usan, qué tipos hay, cómo se configuran y una introducción a las apps creadas por Esri.
Funcionamiento interno Analizaremos qué son los items, qué contienen, cómo se pueden gestionar usando API REST y diferencias en cuanto a arquitectura entre las apps de Esri y las personalizadas.
Crear tus propias apps configurables Repasaremos en qué casos es recomendable usarlas, las ventajas que ofrecen y los requisitos para poder incorporarlas al flujo de trabajo de la organización.
Apps configurables personalizadas Veremos: algunos ejemplos de este tipo de apps, Pros & Cons además de cómo crearlas y configurarlas. Por otro lado analizaremos sus datos y metadatos, repasaremos herramientas y recursos para crealas, y por último recordaremos qué tecnologías podemos usar para desarrollarlas.
